### 2. 问题解释

**为什么会有临时版本？**

你电脑上同时存在两个 OpenClaw：

1. **正常版（2026.3.2）**：通过 `pnpm install -g openclaw` 安装的稳定版本
2. **临时版（2026.3.3）**：从 `/tmp/openclaw-build` 目录直接运行的开发版本

**临时版是怎么来的？**
- 可能是之前手动下载了 GitHub 源码
- 在 `/tmp/openclaw-build` 目录里直接运行
- 这个版本比官方版更新（3.3 vs 3.2），但**不是稳定发布**
