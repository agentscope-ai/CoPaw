## 五、收益最大化专项策略

### 1. 智能择时系统
```
输入：大盘估值 + 情绪指标 + 资金流向
输出：仓位建议（满仓/重仓/半仓/轻仓/空仓）
```

### 2. 基金筛选模型
```
维度：收益排名 + 夏普比率 + 最大回撤 + 规模 + 经理稳定性
输出：五星基金池 + 观察基金池 + 剔除基金池
```

### 3. 组合优化引擎
```
输入：持仓基金 + 风险承受能力
输出：最优配置比例 + 再平衡提醒
```

### 4. 预警系统
- 基金净值异常波动（±3%）
- 基金经理变更
- 市场重大政策变化
- 持仓基金评级下调
