# Copaw 代理协作协议 V1.0

> 所有代理必须遵循此协议进行协作

---

## 一、协作原则

1. **总调度唯一入口** - 所有任务分配必须经过范冰冰
2. **单向委派** - 代理不直接联系其他代理，通过总调度协调
3. **结果汇总** - 支援代理向总调度汇报，总调度汇总给主代理
4. **超时告警** - 任务超时自动上报纪委（周芷若）

---

## 二、协作流程

```
用户任务 → 范冰冰分析 → 确定主代理
                ↓
        主代理接收任务
                ↓
        需要支援？→ 向范冰冰申请
                ↓
        范冰冰分配支援代理
                ↓
        支援代理执行 → 向范冰冰汇报
                ↓
        范冰冰汇总 → 主代理整合 → 最终结果
```

---

## 三、任务申请格式

### 主代理申请支援

```yaml
# 文件：workspace/collab_request_YYYYMMDD_HHMMSS.yaml
request_id: req_xxx
timestamp: 2026-03-08T10:00:00Z
from: 黄蓉
task_type: data_processing
priority: high  # high/medium/low
deadline: 2026-03-08T18:00:00Z
description: |
  需要清洗基金净值数据，计算日收益率
requirements:
  - 输入：raw_fund_data.csv
  - 输出：cleaned_data.csv
  - 字段：日期、净值、日收益率
notes: |
  注意处理节假日空值
```

### 支援代理接收任务

范冰冰将任务转为：

```yaml
# 文件：workspace/collab_task_xxx.yaml
task_id: task_xxx
original_request: req_xxx
from: 范冰冰
assign_to: 小龙女
task_type: data_processing
deadline: 2026-03-08T18:00:00Z
description: |
  [来自黄蓉的请求] 清洗基金净值数据...
```

### 支援代理完成汇报

```yaml
# 文件：workspace/collab_complete_xxx.yaml
task_id: task_xxx
completed_at: 2026-03-08T12:00:00Z
by: 小龙女
status: success  # success/partial/failed
result:
  file: /path/to/cleaned_data.csv
  records: 365
  issues: none
summary: 已完成数据清洗，共365条记录
```

## 六、超时与告警

| 优先级 | 默认时限 | 超时处理 |
|--------|----------|----------|
| high | 4小时 | 告警+升级 |
| medium | 24小时 | 提醒 |
| low | 72小时 | 记录 |

**超时自动上报周芷若（纪委）**

---

## 七、文件存放规范

协作相关文件统一放在各代理 workspace/collab/ 目录：

```
workspace/
├── collab/
│   ├── incoming/      # 接收到的任务
│   ├── outgoing/      # 发出的请求
│   ├── completed/     # 已完成的任务
│   └── archive/       # 归档
```
