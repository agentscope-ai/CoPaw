## 🚀 快速开始贡献（5 分钟）

想要快速贡献但不知道从哪里开始？按照以下步骤，5 分钟内完成你的第一个贡献！

### 第一步：选择任务（1 分钟）

适合新手的任务类型：
- **文档改进**：修正错别字、改进格式、添加示例
- **翻译更新**：改进中文/英文/日文文档的翻译质量
- **Good First Issues**：查看 [Issues](https://github.com/agentscope-ai/CoPaw/issues) 中标记为 `good first issue` 的问题

### 第二步：Fork 并编辑（2 分钟）

1. 在 GitHub 上点击任意文档文件（如 `README_zh.md`）
2. 点击右上角的 **编辑此文件** 按钮
3. 进行你的改进（修正错字、添加内容等）

### 第三步：提交更改（1 分钟）

1. 在页面底部填写提交信息，格式：`docs: 简短描述你的更改`
2. 选择 **Create a new branch for this commit and start a pull request**
3. 点击 **Propose changes**

### 第四步：完成 PR（1 分钟）

1. 在 PR 页面确认更改无误
2. 点击 **Create pull request**
3. 等待维护者审核（通常 24 小时内回复）

### ✅ 提交信息示例

```
docs(readme): 添加快速开始贡献指南
fix(typo): 修正 CONTRIBUTING_zh.md 中的错别字
docs(faq): 添加常见问题解答
```

### 📌 提示

- 小改动可以直接提交 PR，无需事先创建 issue
- 文档类 PR 通常会在 24 小时内审核
- 不确定？在 [Discussions](https://github.com/agentscope-ai/CoPaw/discussions) 中提问！
